"# Emotion-Detection-Testing-Stub" 

First seperate the backend folder into the frontend folder. Put the frontend folder to the Flutter project(if applicable).

To properly able to run the code succesfully you need to install the required requirements.
For the backend u need to install the requirements.txt

Instructions for running the code.
1. Open your file manager and find  the hati_env folder, copy the folder path.
2. Go to the backend folder project path then open the cmd there. Paste this following line of code:
> python -m venv (substitute this by the hati_env folder path)
> (substitute this by the hati_env folder path)\Scripts\activate
3. After that type "python main.py" in the cmd.
4. Next, open your Android Studio application and open the frontend code. Change the base url into your own ipconfig if you are planning to use your real phone for debugging. 
5. After that u can now run the code after choosing your phone as the debugging device.
